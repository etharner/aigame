## Install problem
  * Run `git submodule init`
  * Run `git submodule update`
  * Switch to cats-judge submodule folder
  * Run `perl install.pl`
  * Switch to task folder again
  * Copy `..\config.xml` to `..\cats-judge` folder (overwrite, if exists)
  * Run `j_install.bat`

## Running problem
```
j_run.bat
```
